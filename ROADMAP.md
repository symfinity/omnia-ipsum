## Details

### 0.1.x — First public release

Text, image, audio, and video placeholder generators

### 0.2.x — Generator polish

Broader Faker coverage, provider docs, performance fixes

### 0.2.x - Rich placeholder content

Complete content snippets like Markdown, HTML snippets, Articles, Comments, Reviews, Testimonials, FAQs, Tables, JSON payloads

### 0.3.x - Structured fake data

Complete domain objects like User profiles, Products, Companies, Addresses, Teams, Events, Blog posts, Shop catalog, API fixtures

### 0.4.x - AI-ready content

Fake content but not as lorem ipsum like realistic paragraphs, developer text, legal text, marketing copy, error messages, changelogs, release notes, documentation samples

### 0.5.x - Content relationships

Linked data like Author → Posts, Product → Reviews, User → Orders, Company → Employees, Categories, Tags, Media references

### 0.6.x - Localization

Translated lorem, localized Faker, locale presets, country datasets, currencies, phone formats, addresses

### 0.7.x - Media generation

More than static URLs like SVG placeholders, Open Graph images, QR codes, Charts, PDF placeholders, Avatar themes, Blur placeholders (LQIP)

### 0.8.x - Project presets

One command loads the demo data with severeal presets for Blog, Shop, CMS, SaaS, Dashboard, CRM, Admin

### 0.9.x - Production polish

API freeze, performance improvements, documentation, cookbook, migration guide, extension API, testing matrix

### 1.0.x — Stable API

Semver-stable generator interfaces, handbook and configuration
